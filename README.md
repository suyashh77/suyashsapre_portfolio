### Update Personal Information

Edit the HTML files to update:
- `index.html` - Name, tagline, intro text
- `projects.html` - Project details, videos, links
- `experience.html` - Work history, rotations
- `contact.html` - Contact details

### Change Color Theme

Edit `styles.css` CSS variables:

```css
:root {
  --accent-primary: #0ea5e9;  /* Change primary accent */
  --accent-secondary: #d946ef; /* Change secondary accent */
  /* ... more variables */
}
```

### Modify Layout

- Adjust spacing in `--section-padding`
- Change container width in `--container-width`
- Modify grid layouts in respective sections
